# Astra Linux Security Checker & Toolkit

Набор интерактивных инструментов (TUI) на базе `whiptail` для автоматизации аудита безопасности, сканирования уязвимостей и администрирования операционных систем семейства Astra Linux (в первую очередь Astra Linux Special Edition).

Инструмент разработан для упрощения процессов аттестации систем, настройки мандатного контроля доступа (MAC), управления локальными репозиториями и развертывания антивирусного ПО в изолированных (закрытых) контурах.

---

## 🚀 Основные возможности

### 1. 📊 Комплексный аудит безопасности и снятие скриншотов
- Полная проверка системных параметров ядра (`sysctl`), конфигурации PAM-модулей, сетевого экрана (`UFW`/`iptables`), SSH-сервера и активных сетевых портов.
- Аудит специфичных настроек безопасности Astra Linux (Parsec MAC/MIC, состояние режима ОРД).
- Автоматический запуск Fly/KDE-приложений панели управления и снятие скриншотов настроек параметров (с использованием утилиты `scrot` и авторизацией X-сервера от имени root).
- Сборка интерактивного, структурированного и адаптивного графического отчета в формате HTML (`report/security_report.html`).
- **Контроль соответствия классу защищенности (РД АС / РД СВТ)**: Оценка настроек безопасности системы на соответствие требованиям заданных классов (3Б, 2Б, 1Д, 1Г, 3А, 2А, 1В, 1Б, 1А). Параметры (длина/история паролей, ограничения попыток входа, таймауты, SecDel, SwapWiper, console-lock, интерпретаторы и др.) считываются динамически из настраиваемого конфигурационного файла `service/compliance_standards.conf`. Подробный текстовый отчет сохраняется в файл `report/compliance_report_[КЛАСС]_[ДАТА_ВРЕМЯ].txt`.

### 2. 🔍 Сканирование уязвимостей OpenSCAP (ScanOval)
- Автоматический поиск архивов OVAL-определений ФСТЭК России на подключенных накопителях или их загрузка напрямую из БДУ ФСТЭК (при наличии интернет-соединения).
- Запуск сканирования через OpenSCAP.
- Преобразование стандартного XML-отчета OpenSCAP в наглядный русскоязычный графический HTML-отчет (`report/scanoval/scanoval-report.html`) с помощью специализированного Python-парсера.

### 3. 🛡️ Установка и обслуживание антивирусного ПО (KESL / Dr.Web)
- **Автоматический поиск**: Сканирование всех съемных накопителей и локальной папки проекта на наличие установочных пакетов (`kesl*.deb`, `kesl-gui*.deb`, `drweb*.run`), баз обновлений и ключей лицензий.
- **Интеллектуальная установка**:
  - Установка Kaspersky Endpoint Security (с автоматической генерацией файла ответов `kesl-setup.pl`, автоопределением графической оболочки и поддержкой оффлайн-режима).
  - Установка Dr.Web Security Space (.run установщик в тихом режиме).
- **Офлайн-обновление баз**: Копирование и настройка локального каталога обновлений баз для KESL и Dr.Web без подключения к сети.
- **Активация лицензии**: Автоматическое обнаружение файлов лицензий (`*.key`) на дисках и флешках, выбор из списка или ручной ввод пути. Для Dr.Web производится копирование ключа в надежное локальное хранилище (`/etc/opt/drweb.com/drweb32.key`) и перезапуск демона.
- **Унифицированная обратная связь**: На экран выводится только сообщение об успехе или подробное текстовое окно с логами в случае ошибок.

### 4. 📂 Управление пользователями и мандатным контролем доступа (MAC/MIC)
- **Настройка мандатных уровней доступа (MAC)**: Интерактивный мастер изменения максимального уровня доступа в `/etc/parsec/mac_levels` с возможностью выбора именования по ГОСТ или ввода пользовательских названий, а также автоматическим перезапуском службы Parsec.
- **Создание пользователей с мандатными метками**: Парсинг уровней конфиденциальности системы, валидация ФИО и должности, автоматическое формирование мандатной записи в `/etc/parsec/macdb/<UID>` и установка пароля с проверкой сложности по правилам PAM.
- **Создание и разметка документов**: Автоматическое создание древа каталогов пользователей в `/home/Документы` и разметка папок по мандатным уровням доступа (например, «Открыто», «Секретно») с помощью утилиты `pdpl-file`.
- **Быстрое и безопасное удаление**: Корректное завершение всех процессов пользователя (`pkill`), удаление домашней директории, мандатной записи Parsec и документов.

### 5. 🛠️ Менеджер локального репозитория (Repo Manager)
- Быстрое монтирование установочного ISO-образа или диска Astra Linux.
- Временное комментирование внешних сетевых репозиториев для предотвращения зависаний APT при отсутствии интернета (с автоматическим восстановлением после завершения работ).
- Подключение ISO как доверенного локального источника (`file:/...`) для быстрой оффлайн-установки зависимостей (`openscap-scanner`, `unzip` и т.д.).

## 🛠️ Системные требования

- **Операционная система**: Astra Linux Special Edition (очередные обновления 1.6, 1.7, 1.8 и др.) или Debian/Ubuntu (для базовых функций аудита).
- **Права доступа**: Запуск строго от имени суперпользователя (`root` / `sudo`).
- **Основные утилиты (устанавливаются автоматически через Repo Manager)**:
  - `whiptail` (для работы интерактивного TUI-интерфейса).
  - `openscap-scanner` (для OVAL-сканера).
  - `scrot` (для автоматического снятия скриншотов).

## 🚀 Быстрый старт

1. Склонируйте репозиторий или перенесите файлы на целевую машину (например, через USB-накопитель).
2. Перейдите в каталог проекта и сделайте главный скрипт исполняемым:
   ```bash
   chmod +x hub.sh
   ```
3. Запустите центральный хаб управления:
   ```bash
   sudo ./hub.sh
   ```

## 📂 Структура проекта

```
.
├── hub.sh                      # Главный централизованный хаб управления (TUI)
├── readme.txt                  # Краткая инструкция по запуску
├── about.txt                   # Техническая справка по работе модулей "под капотом"
├── scripts/
│   ├── security_checker.sh     # Аудит безопасности и съемка скриншотов параметров
│   ├── scanoval_checker.sh     # OVAL-сканер уязвимостей ФСТЭК (OpenSCAP)
│   ├── antivirus_installer.sh  # Установка, обновление и активация лицензий KESL / Dr.Web
│   ├── repo_manager.sh         # Управление локальным ISO-репозиторием APT
│   ├── hub_sys.sh              # Настройка мандатного доступа и управление системой (подменю)
│   ├── Создание пользователей.sh # Добавление учетных записей и настройка Parsec MACDB
│   ├── Создание папок для пользователей.sh # Генерация мандатной иерархии папок документов
│   └── Удаление пользователя.sh # Полная очистка данных и удаление аккаунта
├── service/
│   ├── report_generator.sh     # Сборщик основного HTML-отчета аудита
│   ├── generate_custom_report.py # Парсер XML-результатов OpenSCAP в красивый TUI HTML
│   ├── screenshot_apps.list    # Список графических утилит для автоскриншотов
│   └── compliance_standards.conf # Настройки эталонов соответствия классам защищенности РД АС/СВТ
└── report/                     # Каталог сохранения всех отчетов и скриншотов
```
