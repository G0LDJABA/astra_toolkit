========================================================================
             ИНСТРУКЦИЯ ПО ЗАПУСКУ И ИСПОЛЬЗОВАНИЮ
========================================================================

Данный программный комплекс предназначен для аудита безопасности систем Linux.
Все операции должны производиться от имени суперпользователя (root).

------------------------------------------------------------------------
ПОДГОТОВКА И ПРАВА ДОСТУПА
------------------------------------------------------------------------
Перед первым запуском убедитесь, что центральный хаб управления имеет
права на исполнение:
  $ chmod +x hub.sh

(Права на исполнение для остальных скриптов в папке "scripts/" будут
установлены автоматически при их выборе в меню).

------------------------------------------------------------------------
ЗАПУСК ИНТЕРАКТИВНОГО МЕНЮ (ХАБА)
------------------------------------------------------------------------
Запустите центральную панель управления командой:
  $ sudo ./hub.sh

После запуска на экране появится меню с выбором действий:

1) Аудит и скриншоты
   - Выполняет комплексную проверку системы, включая контроль соответствия требованиям РД АС и РД СВТ для заданного класса защищенности.
   - Считывает эталонные требования из настраиваемого файла `service/compliance_standards.conf`.
   - Снимает скриншоты окон параметров (если запущено в графической среде).
   - Результаты сохраняются в:
     * report/security_report.html (интерактивный графический отчет)
     * report/installed_software.txt (список установленного ПО)
     * report/screenshots/ (папка со снимками экранов)

2) ScanOval сканирование
   - Выполняет поиск базы OVAL на внешних накопителях или скачивает ее.
   - Запускает проверку через OpenSCAP.
   - Результаты сохраняются в:
     * report/scanoval/scanoval-report.html (графический отчет)
     * report/scanoval/scanoval-results.xml (XML-результаты)

3) Установка антивирусного ПО
   - Предоставляет TUI-меню выбора между установкой, офлайн-обновлением баз и активацией лицензии для Kaspersky Endpoint Security и Dr.Web Security Space.
   - Поддерживает автоматический поиск установочных пакетов, каталогов баз и файлов лицензий (*.key) на съемных накопителях.

4) Работа с системой (настройка мандатных уровней и управление пользователями)
   - Запускает интерактивный мастер настройки максимального мандатного уровня доступа (MAC) в системе (файл /etc/parsec/mac_levels) с поддержкой ГОСТ или ручного ввода названий уровней, а также перезапуском службы Parsec.
   - Открывает вспомогательное подменю для работы с пользователями:
     * Создание пользователей с мандатными метками (Parsec MACDB).
     * Разметка персональных каталогов документов по мандатным уровням доступа (pdpl-file).
     * Безопасное удаление пользователя (завершение процессов, очистка домашнего каталога, мандатных записей и документов).

5) Инструменты
   - Запускает менеджер локального репозитория `repo_manager.sh`.
   - Позволяет смонтировать установочный образ ISO или диск Astra Linux.
   - Автоматически подключает его как локальный репозиторий в APT.
   - Позволяет установить необходимые пакеты (openscap-scanner, openscap-common, unzip) в автономном режиме.
   - Позволяет размонтировать и удалить запись репозитория после завершения работы.

0) Выход
   - Завершает работу хаба управления.
